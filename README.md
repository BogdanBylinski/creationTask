# File Reader App (CSV / JSON / XML)

Simple PHP app that lets you upload a file (CSV, JSON, or XML) and shows the data in a responsive HTML table. No frameworks, no Composer. PSR-4 autoloader, clean MVC-ish structure.



## Quick start (local)
Option A — PHP built-in server (fastest):
```bash
php -S 127.0.0.1:8080 -t public
```
Open http://127.0.0.1:8080 in your browser.
